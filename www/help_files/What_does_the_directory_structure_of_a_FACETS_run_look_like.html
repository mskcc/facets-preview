What does the directory structure of a FACETS run look like

Facets Preview is currently set up to read FACETS folders in legacy format, which means when generating FACETS data using the FACETS wrapper, the --use-legacy and --everything flags should be included. The following is an example of what an institutional directory might have in terms of files and structure.

The above example has two fits, default and alt-dipLogR fit. The base directory requires facets_qc.txt, facets_review.manifest, and a counts pileup file. There are variations on this structure, but this is a standard example.
